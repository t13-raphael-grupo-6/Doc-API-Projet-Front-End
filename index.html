Documentação API Projeto Front-End

URL DA API: https://api-project-front-end-entrela.herokuapp.com

Criando Usuário

Metodo: POST

Endpoint: /register

Exemplo de corpo a ser enviando para requisição

Unicos campos obrigatorios são EMAIL E SENHA, então podem colocar os inputs necessarios

Exemplo de resposta

Na resposta temos duas informações a mais ID e TOKEN



Login

Metodo: POST

Endpoint: /login

Exemplo de corpo a ser enviando para requisição

É necessário HEADER: Content-Type application/json

Email e senha são obrigatorios

Exemplo de resposta



Criando Necessidades

Metodo: POST

Endpoint: /needs

É necessário o TOKEN da casa beneficente logada

É necessário HEADER: Content-Type application/json


Exemplo de corpo a ser enviando para requisição

No corpo a ser enviado é necessário o id da casa logada no campo "userId"

Exemplo de resposta

O ID retornado é o id da Necessidade criada



Pesquisando todos os usuários

Metodo: GET

Endpoint: /users

Exemplo de resposta



Pesquisando todas as Necessidades

Metodo: GET

Endpoint: /needs

Exemplo de resposta



Pesquisando Necessidades por ID

Metodo: GET

Endpoint: /needs?userId=ID DO USUÁRIO

Exemplo de resposta
